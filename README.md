# Movie Recommendation System

This project is a simple Movie Recommendation System built using Python and Machine Learning.  
It suggests similar movies based on genre similarity.

## Features
- Select a movie from the dropdown
- Get recommended movies instantly
- Built using a content-based filtering approach

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Dataset
MovieLens dataset containing 100k+ movie ratings.

## How to Run the Project

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the app

streamlit run app.py
